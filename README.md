# The Global Times – Newspaper Layout

## Description
A **responsive newspaper webpage** built with **HTML and CSS Grid**, featuring:

- Title spanning the top  
- Feature article with cover image  
- Secondary article  
- Three small articles in the last row  
- Responsive layout for desktop and mobile  

## Technologies
- HTML5  
- CSS3 (Grid, Flexbox)  
- Responsive design with media queries
