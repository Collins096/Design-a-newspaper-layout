📄 The Global Times – Newspaper Layout
Description

This project is a responsive newspaper webpage layout built using HTML and CSS Grid. It demonstrates modern web design techniques such as:

Semantic HTML5 elements (<header>, <article>, <figure> etc.)
CSS Grid with grid-template-areas for precise layout control
Responsive design with media queries
Polished typography, spacing, and visual hierarchy

The layout includes:

A main title/header spanning the top
A feature article spanning two columns with a cover image
A secondary article spanning two columns
Three smaller articles in the last row
Images that fit perfectly within their grid areas

This project is a great reference for learning CSS Grid and creating clean, modern newspaper-style layouts.

🛠 Features
Fully responsive layout for desktop, tablet, and mobile
CSS Grid with named grid areas for easy content placement
Clean typography and spacing for readability
Cover image with caption, properly sized with max-width: 100%
Modern card-like design for all articles
🖥 Technologies Used
HTML5 – semantic structure
CSS3 – grid layout, flexbox, and styling
Responsive Design – media queries for different screen sizes
